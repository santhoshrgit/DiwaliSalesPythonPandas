Introduction

The dataset used in this project is the Diwali Sales Data, which contains transactional records of customer purchases during the Diwali season. It includes various details such as customer demographics, product categories, purchase amounts, and payment methods. This dataset is useful for analyzing sales and order details

Libraries Used

To analyze and visualize the data, the following Python libraries are used:
Pandas for data manipulation and analysis
NumPy for numerical computations
Matplotlib & Seaborn for data visualization
